Welcome to the docs! You can start writing documents in `/content/docs`.

## What is Faiz UI?

Faiz UI is a React component library that provides a set of reusable freehand components for building web applications. It is built on top of Tailwind CSS and React.

## Installation

```package-install
npm install @faiz-ui/react @faiz-ui/theme
```
