# Alert

Alert component is used to display important messages or feedback to users. It supports various styles, colors, and customization options to match different use cases and severity levels.

## Usage

```tsx
import { Alert } from '@faiz-ui/react'
<Alert>This is an alert message</Alert>
```

<Alert>This is an alert message</Alert>

## Without Icon

You can hide the icon by setting the `hideIcon` prop to `true`.

```tsx
<Alert hideIcon color="primary">Alert without icon</Alert>
```

## With Action

You can add action buttons to the alert.

```tsx
<Alert color="primary" action={<button className="px-2 py-1 text-sm bg-blue-100 dark:bg-blue-900 rounded-md">Action</button>}>Alert with action button</Alert>
```

## Controlled Visibility

You can control the visibility of the alert using the `isVisible` prop.

```tsx
const [isVisible, setIsVisible] = useState(true)

<button onClick={() => setIsVisible(!isVisible)}>
  {isVisible ? 'Hide Alert' : 'Show Alert'}
</button>

<Alert color="primary" isVisible={isVisible} onClose={() => setIsVisible(false)}>
  Alert with controlled visibility
</Alert>
```

## Props

| Prop         | Type                                                                       | Default     | Description                              |
| ------------ | -------------------------------------------------------------------------- | ----------- | ---------------------------------------- |
| color        | `'primary' \| 'secondary' \| 'success' \| 'warning' \| 'danger' \| 'info'` | `'primary'` | The color variant of the alert           |
| variant      | `'solid' \| 'outline' \| 'ghost' \| 'light' \| 'flat' \| 'sketchy'`        | `'solid'`   | The style variant of the alert           |
| radius       | `'none' \| 'sm' \| 'md' \| 'lg' \| 'xl' \| 'full'`                         | `'md'`      | The border radius of the alert           |
| hideIcon     | `boolean`                                                                  | `false`     | Whether to hide the icon                 |
| icon         | `ReactNode`                                                                | -           | Custom icon to display in the alert      |
| action       | `ReactNode`                                                                | -           | Action component to display in the alert |
| isVisible    | `boolean`                                                                  | `true`      | Whether the alert is visible             |
| onClose      | `() => void`                                                               | -           | Callback fired when the alert is closed  |
| customStyles | `string`                                                                   | -           | Custom styles for the alert              |
